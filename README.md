# Dotter

Dot-file management program in pure python, no dependencies

## Intended usage

Dotter is meant to save linux/unix dot-configuration files in one central
location to make them easily available for version management-systems.
All registered files are symlinked into that location. This makes working
on files as well as re-setting up a system much more easy.

## Usage

- Run dotter in a .config-directory
- It will start as a file-browser
